# 🛒 MyShop

MyShop is a beginner full-stack e-commerce project built using FastAPI, Python, HTML, CSS, and JavaScript.

## Features

* User Signup
* User Login
* JWT Authentication
* Add Products
* Delete Products
* Product Listing
* Category Filtering
* FastAPI Backend
* Responsive Frontend

## Tech Stack

### Backend

* Python
* FastAPI
* JWT Authentication
* SQLite Database

### Frontend

* HTML
* CSS
* JavaScript
